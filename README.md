# 🌦️ Mausam Heram

**Mausam Heram** (मौसम हेरम्) means **"Let's Look at the Weather"** in Nepali. It is a modern desktop weather application built with Python and Tkinter that provides real-time weather information for cities around the world.

Using the OpenWeatherMap API, the application displays current temperature, weather conditions, humidity, wind speed, visibility, and feels-like temperature through a clean and user-friendly interface.

## ✨ Features

* 🌍 Search weather by city name
* 🌡 Real-time temperature updates
* ☁ Current weather conditions
* 💧 Humidity information
* 💨 Wind speed details
* 🌡 Feels-like temperature
* 👁 Visibility status
* 🕒 Live date and time display
* 🌤 Dynamic weather icons
* 🎨 Modern dark-themed UI
* ⚡ Fast and responsive performance

## 🛠 Built With

* Python
* Tkinter
* Requests
* Pillow (PIL)
* OpenWeatherMap API

## 🚀 Installation

```bash
pip install requests pillow
```

Run the application:

```bash
python weather.py
```

## 📸 Preview

A sleek weather dashboard designed to provide essential weather information in a simple and elegant way.

## 🎯 Purpose

Mausam Heram was created to make checking weather conditions simple, fast, and visually appealing while helping the developer learn API integration, multithreading, and modern GUI development in Python.
